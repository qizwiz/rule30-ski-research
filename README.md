# Rule30 SKI Research

Standalone research repo extracted from `openclaw-openclaw-full`.

## Contents
- `papers/`: canonical paper artifacts
- `lean_proofs/`: Lean formalization files
- `research/`: funding and related-work bundle
- `data/`: measured datasets
- Core scripts in repo root for reproducibility

## Quickstart
- `python3 ski_ca_analyzer.py`
- `python3 ski_rule30_center.py --gen 5`
- `python3 extract_basins.py`
- `python3 analyze_growth.py`
- `lean lean_proofs/InterBasin.lean`
